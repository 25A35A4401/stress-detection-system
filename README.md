📊 Stress Detection System
📌 Overview
The Stress Detection System is a web-based application that analyzes a user's daily lifestyle habits and predicts their stress level. It classifies stress into Low, Medium, or High and provides useful suggestions to improve mental well-being.

🚀 Features
Simple and user-friendly interface
Detects stress using daily inputs
Classifies stress into 3 levels
Provides personalized suggestions
No hardware required
Instant results

🧠 Input Parameters
Sleep Duration (hours)
Work/Study Load (hours)
Screen Time (hours)
Pressure Level (1–5)
Emotional Well-being (1–5)

⚙️ Technologies Used
Frontend: HTML, CSS
Backend: Python
Framework: Flask
Templating: Jinja

🔄 How It Works
User enters daily lifestyle details
System processes the input
Calculates stress score using rule-based logic
Displays stress level (Low / Medium / High)
Provides suggestions to reduce stress
